### Description
This ModelBrick from the article, "Sniffers, buzzers, toggles and blinkersâ€¦" describes a system containing both positive and negative feedback, resulting in oscillations. The response element (R) is created in an autocatalytic process, and then it promotes the production of an inhibitor, X, which speeds up R removal. First, R builds up, then comes X to force R back down, then X disappears and R can rise again.
